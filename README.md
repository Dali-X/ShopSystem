# ShopSystem

这个系统使用PyQt5，用于商店日常的管理

有下列功能：

- 密码登录
- 商品\手法 的购买
- 打印小票
- 商品的添加与修改
- 用户充值
- 数据统计
- 员工消息增加（待开发）
